# Dictation Buddy

Dictation Buddy is a self-hosted, lightweight web application designed to help children (and adults) practice dictation. It automatically splits audio files (MP3, MP4, M4A) into sentences using silence detection, allowing you to play, repeat, and navigate through a dictation exercise sentence by sentence.

## Features
Auto-Splitting: Automatically detects sentences based on silence gaps.

Local Privacy: Runs entirely on your local machine or home server (no cloud uploads).

Format Support: Plays .mp3, .mp4, and .m4a files.

Touch Friendly: Large, easy-to-tap buttons for iPad/Tablet users.

Tuning: Adjust silence thresholds and pause lengths on the fly to fix splitting errors.

## How to use
Simply clone this project and run `docker-compose up -d`, it would build the image and run the container
